# Level Devil

Level Devil is a fast-paced platformer game where you must navigate through 10 devilishly designed levels, each filled with unique challenges and evil surprises. Can you escape the devil's realm?

> This project was created using Copilot for enhancing multiple stages of the project and to increase the overall beauty of the game

## How to Play

- Use **Arrow Keys** or **WASD** to move
- Press **Space** to jump (hold for higher jumps)
- Press **Enter** to start/restart the game
- Reach the red door to complete each level
- Collect golden coins for extra points
- Watch out for the Devil's Tricks!

## Features

### Core Mechanics
- Precision platforming controls
- Adjustable jump heights
- Player trail effects
- Lives system with heart display
- Score tracking and death counter
- Particle effects for all actions

### Level Elements
- Regular platforms
- Disappearing platforms
- Falling blocks that crumble when approached
- Fake platforms that vanish when touched
- Moving and static spikes
- Collectible coins with varying values
- Teleporters with cooldown system
- Speed boost power-ups
- Reverse gravity zones

### Devil's Tricks
- Random spike spawns
- Reversed controls
- Invisible spikes
- Mirror mode
- Speed changes
- Gravity manipulation
- Ultimate chaos mode
- Apocalyptic finale

### Special Effects
- Screen shake on death
- Warning system for traps
- Dynamic particle effects
- Glowing visual effects
- Sound effects for all actions
- Pulsing UI elements

### Level Design
- 10 unique levels with increasing difficulty
- Each level has its own name and theme
- Progressive challenge system
- Checkpoint system at each level
- Final boss level: "Satan's Final Test"
